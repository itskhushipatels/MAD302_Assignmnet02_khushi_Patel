Description
An Android app that shows a list of products using RecyclerView. Tapping a product opens a detail screen with full information, passed between screens using Intent.
How to Run
1. Clone the repo
2. Open in Android Studio
3. Run on an emulator or device (API 24+)
Features
RecyclerView with 8 products
Adapter and ViewHolder pattern
Lambda click handling
Intent navigation to detail screen
Null-safe data passing between activities
